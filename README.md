# XGBoost-SIMEX-Comparison
A QGIS processing script written in Python to compare land cover classification output by an XGBoost model to SIMEX.

## Installation
- Download the processing script file (xxx.py)
- Drag the file into your QGIS active profile folder in the following location:
  Roaming > AppData > [user profile] > QGIS > QGIS3 > profiles > default > processing > scripts
- Now load up a blank new QGIS project
- Open the processing toolbox under 'Processing' > 'Toolbox' or by pressing Ctrl+Alt+T
- Click on the Python 'Scripts' button and select 'Add Script to Toolbox'
- The processing script should now appear under Scripts > scripts > Compare XGB and SIMEX within the toolbox menu
